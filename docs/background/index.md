# Background

The background section now explains the biological and computational problem that AIMS-LEAF is trying to solve.

The project focus is plant heat stress tolerance, with documentation centered on:

- host response as a multiscale phenomenon
- integration of sequencing and imaging evidence
- model development for prediction rather than one-off data description
- scalable infrastructure for iterative AI/ML workflows

These pages are intended to clarify why the project exists, what kinds of questions it is designed to support, and how the documentation is being reframed during the ongoing schema refactor.
